# otamatone-simulator

## 준비물
Python 3.8버전 또는 그 이상 버전

## 실행 전
1. clone repository
```bash
git clone [리포지토리주소]
```
2. Install required packages
```bash
pip install -r requirements.txt
```
3. Create a virtual environment
```bash
python -m venv venv
```

## 실행 방법
1. Activate the virtual environment
```bash
venv\Scripts\activate
```
2. Run the Application
```bash
cd source
python main.py
```
